Projects

Image Forgery Detection

Developed an Image Forgery Detection system using Convolutional Neural Networks (CNN) and Error Level Analysis (ELA) to identify manipulated or tampered images. With the rise of digital image editing tools, detecting forgeries is crucial in journalism, legal evidence, and digital forensics. The system uses ELA to preprocess images by resaving them and analyzing the error levels introduced during compression. Authentic images exhibit uniform compression artifacts, while altered regions show irregularities. ELA highlights these inconsistencies by creating a visual map of potential manipulation areas. The processed images are then passed through a CNN model, which automatically extracts features and classifies the image as “authentic” or “forged.” The CNN was trained on a labeled dataset of real and manipulated images to ensure reliable detection. The project was implemented using Python, with libraries like OpenCV, PIL, and TensorFlow/Keras. A simple user interface allows users to upload an image and receive immediate feedback on its authenticity. This project demonstrates the power of combining traditional forensic techniques with modern AI to solve real-world problems in image verification and digital security.
